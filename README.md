# 🔐 Password Validator

A simple Python program that validates a user password based on basic input validation rules.

## 📖 Description

This project demonstrates how to use Python built-in functions to validate user input. The program asks the user to enter a password and checks whether it meets the required minimum length.

## ✨ Features

* Accepts password input from the user
* Checks password length for validation
* Demonstrates basic Python input handling
* Beginner-friendly implementation

## 🎯 Learning Purpose

This project was created to practice fundamental Python concepts such as user input, data validation, and conditional statements.
